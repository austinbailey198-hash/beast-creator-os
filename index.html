The concept: "Beast Crate: Back to School"

Mood board notes

  • Matte-black crate, crowbar-open reveal. The unboxing is the content.
  • Feastables + Lunchly co-pack: "fuel for the first day back."
  • Personalized note card w/ creator's @ printed, so it's postable.
  • One "golden ticket" per 100 crates → surprise campaign hook.

Tier structure

  • Tier A · 50 marquee: custom crate, handwritten note, early drop
  • Tier B · 150 core: standard crate, full product set
  • Tier C · 300 niche/emerging: mailer box, hero SKUs
Gifting planner live
Run of show
Wk 1-2

Lock concept + manufacturer

Finalize crate SKU list with supply chain; get sample from packaging vendor; confirm lead time against go-live. Reconcile against real inventory before promising units.

Wk 2-3

Source + score 500 creators

Pull from CRM, run fit model, over-source 20% to absorb no-replies. Legal pre-clears any FTC/gifting-disclosure language.

Wk 3-4

Outreach + address capture

Personalized DMs, secure shipping consent + W-9 where needed, dedupe addresses, flag minors out (18+ only, no exceptions).

Wk 5

Assembly + QC + wave ship

Staggered ship in 4 waves so content doesn't all land same day. Track numbers by tier; hold buffer stock for reships.

Wk 6-8

Capture + measure

Monitor organic posts, log EMV + engagement per creator, ID the top 10% for paid follow-up partnerships.
